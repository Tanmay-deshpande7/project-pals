# ProjectPals 🚀

ProjectPals is a comprehensive, real-time collaboration platform designed specifically for college students and developers to connect, build projects together, and manage tech events. 

## 🌟 Key Features

*   **Real-time Collaboration:** Find teammates, apply for roles, and build future tech together.
*   **Live Chat System:** Integrated real-time messaging between project team members using Firebase.
*   **Dynamic Event Management:** A dedicated Organizer Portal where admins can create events, design custom registration forms, and monitor student applications.
*   **Premium UI/UX:** Built with a modern, glassmorphic dark-mode aesthetic utilizing Tailwind CSS and React.
*   **Serverless Architecture:** Fully powered by Firebase (Authentication, Firestore, Hosting) for incredible speed and zero backend maintenance.

## 🛠️ Technology Stack

*   **Frontend:** React 18, Tailwind CSS, Lucide Icons
*   **Backend / Database:** Google Firebase (Firestore)
*   **Authentication:** Firebase Auth
*   **Hosting:** Firebase Hosting (Multi-site: Main, Admin, Events)

## 📁 Project Structure

The platform is divided into three interconnected applications:

1.  `/public` - The main student-facing application where users can discover projects, chat, and register for events.
2.  `/admin` - A secure portal for administrators to manage overarching platform metrics.
3.  `/events` - A dedicated workspace for event organizers to create, manage, and monitor college hackathons and fests.

## 🚀 Getting Started

This project is deployed live via Firebase Hosting. 

To run locally:
1. Ensure you have Node.js and Firebase CLI installed.
2. Clone the repository.
3. Run `npm run dev` to start the local development server.
4. Run `npm run build` to compile the Tailwind CSS and Babel JavaScript bundles.
